读取目录

获取某目录下全部子目录

ls -F | grep ‘/$’
注：ls -F 显示的目录以/结尾,再用grep筛选出来,-F参数使得ls命令可以在显示子目录的时候在它的文件名之后加上一个斜线(“/”)字符。而文件后面的星号(“*”)字符表示这是一个可执行程序

ls -l | grep ‘^d’
注：ls -l 显示的目录以d开头

ls -ld */

ls -ap | grep ‘/‘
注：ls -p可以给目录加上一个’/’，过滤一下就能只显示目录了,-a显示隐藏文件。

获取某目录下全部文件

ls -F | grep -v ‘/$’
注：grep -v 找相反的，也就是不以/结尾的文件

ls -F | grep -v ‘/$’ | grep ‘.py$’
注：找出文件，且以.py结尾的。
